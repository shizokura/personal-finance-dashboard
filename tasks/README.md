# Tasks

This directory contains task breakdowns organized by PRD (Product Requirements Document).

## Structure

```
tasks/
└── prd-001-single-currency-insights/
    ├── README.md
    ├── phase-1-architecture.md
    ├── phase-2-settings-page.md
    ├── phase-3-insights-page.md
    ├── phase-4-ui-updates.md
    ├── phase-5-migration-storage.md
    └── phase-6-testing.md
```

## Adding New PRDs

When creating tasks for a new PRD:

1. Create a new directory: `tasks/prd-{identifier}-{name}/`
2. Add a `README.md` with overview and implementation order
3. Add phase/task markdown files
